## 웹사이트 디자인

**디자인 방향**: 에디토리얼 매거진 × 금융 인텔리전스 다크모드.

- **타이포그래피** — Fraunces · Noto Serif KR · IBM Plex Sans/Mono
- **컬러** — 잉크 블랙(`#0B0D10`) 바탕에 따뜻한 크림(`#ECE6D8`), 구리(`#D4A574`)·세이지·러스트 액센트
- **애니메이션 배경** — 노이즈 함수 기반 등고선 라인이 마우스에 반응하며 천천히 흐름. 부유 입자 + 그레인 오버레이
- **인터랙티브 요소**
  - 시나리오 카드 클릭 → 단지별 진입가 테이블 실시간 갱신
  - 캔버스 가격 차트 (스크롤 진입 시 라인 드로잉 + 거래량 막대)
  - 체크리스트 ↔ SVG 원형 프로그레스 링 동기화
  - 마우스 추적 광원 헤일로
  - 카운터·바·등장 애니메이션 (IntersectionObserver)

산출물입니다. 진입 트리거는 `suji-apt-analysis-orchestrator` 스킬.
