# 아파트 매매 타이밍 분석

## 언제 사용하는가

- 가격 데이터(`_workspace/01_*`)와 시장 환경 분석(`_workspace/02_*`)이 준비된 상태에서 매매 시점을 판단해야 할 때
- "지금 사도 되는가 / 언제까지 기다리는 것이 좋은가" 같은 질문에 답할 때
- 사용자가 매매 의사결정 체크리스트를 요청할 때

## 핵심 원칙: 시나리오 사고

**단일 정답을 제시하지 않는다.** 부동산 시장은 외생 변수(금리·정책·공급)에 크게 의존하므로, 3개 시나리오로 분기하여 각 시나리오의 발생 조건과 적합한 행동을 제시한다.

### 3 시나리오 구조

| 시나리오 | 가정 | 가격 경로 | 추천 행동 |
|---------|------|----------|---------|
| 베이스 | 현재 추세 지속 | 횡보 또는 완만한 ±5% | 실거주 진입 검토 / 투자는 관망 |
| 낙관 | 금리 인하 + 공급 부족 | 6~12개월 내 +10~20% | 실거주는 조기 진입 / 투자는 진입 검토 |
| 비관 | 고금리 지속 + 공급 과잉 + 정책 강화 | 6~12개월 내 -10~20% | 매수 보류, 매도자는 신속 처분 검토 |

각 시나리오에 발생 확률을 정성 평가하고(예: 베이스 50%, 낙관 25%, 비관 25%), 근거를 명시한다.

## 분석 절차

### 1단계: 사이클 위치 식별

수지구 매매가격 시계열을 보고 현재 위치를 평가:
- **상승 초기** — 거래량 회복, 매도 호가 상승, 매수자 우위 → 매수자 우위
- **상승 말기** — 가격 신고가 갱신, 거래량 정점 후 감소 → 매수 시 추가 상승 여지 제한
- **조정 초기** — 거래량 절벽, 호가만 유지 → 추가 하락 가능성 — 지켜볼 시기
- **조정 말기** — 거래량 점진 회복, 급매 소화 → 매수자에게 유리한 진입 구간
- **횡보** — 거래량 약한 균형, 가격 변동성 낮음 → 트리거를 기다리는 시기

근거: 가격 차트 + 거래량 차트 + 호가-실거래 갭

### 2단계: 변곡점 트리거 식별

미래 가격을 움직일 트리거 이벤트를 나열하고 모니터링 가능한 지표를 명시:

| 트리거 | 모니터링 지표 | 가격 영향 (방향·강도) |
|--------|------------|-----------------|
| 기준금리 인하 시작 | 한은 금통위 발표 | ↑ 강 |
| 가계대출 규제 강화 | 금융위 발표 | ↓ 중 |
| 광교·판교 입주 물량 집중 | 입주 캘린더 | ↓ 약~중 |
| 1기 신도시 특별법 가시화 | 국토부 시행계획 | ↑ 약 (인접 효과) |
| GTX-A 정시 운행 | 개통 일정 | ↑ 중 |

### 3단계: 목적별 매트릭스

**실거주 목적**
- 보유 기간 5년+ 가정 → 단기 변동성보다 입지·생활 적합성 우선
- 자금 여력이 있고 마음에 드는 단지를 발견했다면, 시장 사이클이 "상승 초기" 또는 "조정 말기"일 때 진입 검토
- "최저점 매수"는 사실상 불가능 — 적정가 진입 + 장기 보유 전략 권장

**투자 목적**
- 보유 기간 3~5년 가정 → 시장 사이클과 변곡점 트리거에 민감
- 진입 시점: 조정 말기 + 거래량 회복 시그널
- 매도 시점: 상승 말기 시그널 (신고가 + 거래량 감소)
- 출구 전략과 손절 기준을 사전에 정해두는 것이 핵심

### 4단계: 의사결정 체크리스트

매수 전 점검 항목:
- [ ] 자기자본 비율 (LTV 한도 내 + 여유 자금 6개월 생활비)
- [ ] DSR 한도 내 월 상환액 적정성
- [ ] 보유 비용 (취득세, 재산세, 관리비, 대출 이자) 5년 총액 계산
- [ ] 출구 전략 (장기 보유 / 5년 내 매도)
- [ ] 보유 기간 동안 발생 가능한 정책 리스크 (양도세·종부세 변경 시나리오)
- [ ] 매수 대상 단지 3~5개 비교 (가격·입지·관리 상태)

매도 전 점검 항목:
- [ ] 양도세 계산 (보유·거주 기간, 비과세 요건)
- [ ] 매도 이후 거주지 계획 (이주 비용 포함)
- [ ] 대체 투자처 (현금화 후 자금 활용 계획)

## 분석 시 절대 회피해야 할 표현

- "반드시 오른다 / 반드시 내린다"
- "지금이 마지막 기회"
- "최저점이다 / 최고점이다"
- "○월에 사라" (정확한 시점 단정)

이런 표현은 시장의 본질적 불확실성을 무시한다. 대신 "시나리오 X가 실현될 경우, ○~○월이 진입 적정 구간으로 평가됨"처럼 조건과 범위를 명시하는 어조 사용.

## 분석 시 권장 표현

- "현재 데이터 기준으로는 ○ 시나리오의 확률이 높게 평가됨"
- "○ 트리거가 발생할 경우 시나리오를 재평가 필요"
- "낮은 확신도" / "중간 확신도" / "높은 확신도" 라벨로 결론 강도 표기

## 흔한 함정

- **확증 편향**: 자신의 결론에 맞는 데이터만 강조 → 반대 시나리오도 명시적으로 다루기
- **과거 패턴 맹신**: 2017~2021 상승 사이클이 반복된다는 가정 위험 — 인구·금리·정책 환경 다름
- **단기 뉴스 과반영**: 한 달의 거래량 반등으로 추세 단정 금지
- **개인화 누락**: 같은 시장이라도 자금·목적·세금 상태에 따라 결론이 다름 — 항상 "목적별" 추천 분기
