# 부동산 시장 환경 분석

## 언제 사용하는가

- 가격 데이터만으로는 부족하고 "왜 그런 가격이 형성되었는가" / "앞으로 어떻게 바뀔까"가 필요할 때
- 매매 타이밍을 판단하기 전 시장 사이클을 파악할 때
- 수지구를 포함한 수도권 아파트 시장의 환경적 압력 요인을 정리할 때

## 분석 프레임워크: 4 × 3 매트릭스

요인을 4개 축, 영향 평가는 3개 축으로 분리한다.

**요인 4축**
1. **거시 경제**: 기준금리, 주담대 금리, 환율, 가계대출 규모
2. **정부 정책**: LTV/DSR, 양도세·종부세, 대출 규제, 분양가 상한제
3. **공급·인구**: 전국·수도권 입주 물량, 인구 구조, 가구 분화
4. **지역 특성**: 교통, 학군, 일자리, 정비(재건축/리모델링)

**영향 3축**
1. **방향**: 가격에 ↑(상승 압력) / ↓(하락 압력) / → (중립)
2. **강도**: 약 / 중 / 강
3. **시간 지평**: 단기(6개월) / 중기(1~2년) / 장기(3~5년)

## 거시 환경 분석 항목

| 항목 | 점검 포인트 | 데이터 출처 |
|------|-----------|-----------|
| 기준금리 | 한국은행 현 수준, 향후 6~12개월 전망 (선물·전망 보고서) | 한국은행, 금통위 회의록 |
| 주담대 금리 | 변동·고정 비교, 우대 수준 | KB·한국은행 통계 |
| 가계대출 | 잔액 추세, 신규 취급 추이 | 한국은행 |
| DSR/LTV | 현행 규제와 예고된 변경 | 금융위, 국토부 |
| 양도세·종부세 | 다주택자 세율, 비과세 요건, 보유세 부담 | 기획재정부, 국세청 |
| 분양 정책 | 분양가 상한제, 청약 규제 | 국토부 |

## 지역 특성 분석 — 수지구 체크리스트

**교통**
- 신분당선(강남까지 30분대) — 수지구의 핵심 강점
- 분당선 (구간 정거장)
- GTX-A 영향 (광교/성남 환승 가능성)
- 경부고속도로 신갈IC 접근성
- 향후 GTX-C 연장 등 가능성 (확정/미확정 구분)

**학군**
- 풍덕초, 성복초, 죽전초 등 명문 초등학교
- 수지고·풍덕고 등 학업 성취도
- 죽전·풍덕천 학원가 규모
- 학군 수요는 매매보다 전세 시장에서 더 강하게 반영됨

**일자리**
- 판교 테크노밸리 (10~20분대) — 30대 수요 주축
- 강남(신분당선 직결) — 고소득층 수요
- 광교 R&D 단지

**정비**
- 수지구는 1기 신도시(분당·평촌·일산·중동·산본)에 포함되지 않음 — 특별법 직접 수혜 ✗
- 다만 분당 정비 활성화 시 인접 효과 (이주 수요 유입) 가능성
- 일부 30년 이상 단지의 리모델링 시도

**공급**
- 수지구 내 신규 분양 (제한적, 토지 부족)
- 인접 광교·판교 신축 영향
- 처인구·기흥구 입주 물량 (수지 수요 일부 분산)

## 분석 시 주의

- **방향만 보지 말고 강도 평가**: "금리 인하 가능성"이 있어도 "약한 인하 폭"이면 가격 영향은 미미할 수 있음
- **시점 명시**: 정책·금리는 자주 바뀌므로 "2026년 5월 기준" 등 시점을 분석 시작 부분에 기록
- **상충 요인 인정**: 호재(금리 인하 가능성)와 악재(공급 과잉)가 동시에 존재할 수 있음 — 둘 다 기록하고 우세 판단 근거 제시
- **확정 vs 가능성**: "GTX-C 수지 연장"은 정치적 논의 단계인지, 예타 통과인지, 착공 단계인지 구분. 단계별로 가격 반영 강도가 다름

## 자주 만나는 함정

- **선반영 무시**: 호재는 이미 가격에 반영됐을 수 있음 — 추가 상승 여지보다 "이미 반영됨" 가능성을 먼저 점검
- **단기 뉴스 과대 해석**: 한 달의 금통위 발언으로 추세를 단정하지 말 것
- **지역 외 요인 누락**: 수지만 봐서는 안 되고 광교·판교 신축 공급이 수지 가격에 미치는 영향도 평가
