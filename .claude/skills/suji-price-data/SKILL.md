# 수지구 아파트 가격 데이터 수집

## 언제 사용하는가

- 경기도 용인시 수지구(또는 풍덕천/죽전/상현/성복/신봉/동천)의 아파트 가격 데이터를 수집해야 할 때
- 실거래가·KB시세·전세가율·거래량 시계열을 작성할 때
- 단지별 비교 자료가 필요할 때

## 출처 우선순위

1. **국토교통부 실거래가 공개시스템** (`rt.molit.go.kr`) — 체결 데이터, 가장 신뢰성 높음. 단점: UI가 무거움
2. **한국부동산원 R-ONE / KOSIS** — 지역별 매매가격지수, 전세가격지수
3. **KB부동산 / KB리브온** — 주간 시세, 일반·상위·하위 평균가
4. **네이버부동산 / 호갱노노 / 아실** — 단지별 매물·시세 분석에 유용 (실거래가 + 호가)
5. **직방·다방** — 매물 트렌드 보조 참고
6. 민간 블로그·유튜브는 사실 확인용으로만 참고, 본 자료로 채택 금지

## 데이터 수집 절차

1. **표본 단지 선정**
   - 수지구 6개 행정동에서 각 2~3개 단지 선정
   - 선정 기준: 가구 수 500세대 이상, 거래 빈도 높은 단지 우선
   - 학군·연식·평형이 다양하도록 안배 (예: 신축 1, 10년차 1, 20년차 이상 1)

2. **시계열 깊이**
   - 최근 6개월: 주별 또는 월별
   - 최근 1년: 월별
   - 최근 5년: 분기별
   - 최근 10년: 연도별
   - 이전 사이클 비교가 필요하면 2008·2013·2019·2022년 변곡점 데이터 추가

3. **수집 항목 표 (표준 스키마)**

   | 필드 | 설명 | 출처 |
   |------|------|------|
   | 단지명 | 정식 명칭 | - |
   | 행정동 | 풍덕천1동 등 | - |
   | 준공연도 | YYYY | - |
   | 세대수 | 정수 | - |
   | 평형 | 전용 ㎡ / 분양평수 | - |
   | 시점 | YYYY-MM 또는 YYYY-MM-DD | - |
   | 실거래가 | 만원 단위 | 국토부 |
   | KB시세_일반 | 만원 단위 | KB |
   | 전세가 | 만원 단위 | 국토부/KB |
   | 거래건수 | 월별 건 | 국토부 |
   | 출처URL | 조회 URL | - |
   | 조회일 | YYYY-MM-DD | - |

4. **호가와 실거래가 구분**
   - 네이버부동산·호갱노노 등에서 보이는 가격은 호가(매도 희망가) — 실제 체결가가 아님
   - 호가는 시장 분위기 참고용으로만, 본 자료에는 실거래가를 채택
   - 호가-실거래 갭이 클수록 매수자-매도자 힘 차이가 큼 (지표로 활용 가능)

5. **거래량 동시 수집**
   - 가격이 떨어졌는데 거래량도 거의 없으면 "거래 절벽 + 호가 유지" 상태
   - 가격이 올랐고 거래량도 늘었으면 "추세 형성" 가능성
   - 거래량 분석 없는 가격 분석은 왜곡 가능성 높음

## 수지구 주요 표본 단지 (참고용 시작점)

> 아래는 일반적으로 자주 언급되는 단지의 예시이며, 실제 수집 시 최신 정보로 검증해야 한다.

- **풍덕천동**: 수지래미안1차, 풍덕천현대2차, 신정마을 단지군
- **죽전동**: 죽전한라비발디, 새터마을 단지군
- **상현동**: 광교상현, 상현마을 단지군
- **성복동**: 성복역롯데캐슬, 성복자이
- **신봉동**: 신봉센트레빌
- **동천동**: 동천자이

## 결과 파일 형식

`_workspace/01_price_collector_data.md`에 다음 구조로 저장:

```markdown
# 수지구 아파트 가격 데이터 (조회일 YYYY-MM-DD)

## 1. 수지구 전체 가격 지수
- KB 매매가격지수 추이 (월별 표)
- 한국부동산원 지수 (비교)

## 2. 거래량 추이
- 월별 매매 거래 건수 표

## 3. 단지별 실거래가
- 단지별 표 (단지명·평형·시점·가격·거래량)

## 4. 전세가율
- 단지별 전세가율 추이

## 5. 공급 자료
- 최근 입주·예정 단지 (수지구 + 인접 광교/판교)

## 6. 출처 및 한계
- 사용한 URL 목록
- 데이터 부족 구간 명시
```

## 데이터가 부족할 때

- 특정 단지의 실거래 빈도가 낮아 가격 추이를 못 그리면 → 인근 유사 단지로 대체하고 명시
- 5년 이전 데이터가 거의 없으면 → 한국부동산원 지수로 대체하고 "단지 단위 자료 부재" 명시
- 출처가 서로 다른 수치를 보일 때 → 모두 병기하고 차이 사유 추정 (예: 평균 산정 방식 차이)

## 주의 — 환각 방지

- 실거래가는 반드시 출처를 확인하고 인용. 추정·짐작으로 수치 작성 금지
- 데이터를 찾지 못했을 때 "찾지 못함"이라고 명시하는 것이 가짜 수치를 만드는 것보다 가치 있음
- 조회 시점을 반드시 기록 — 부동산 시세는 며칠 단위로 변하므로 시점 없는 수치는 신뢰도 낮음
